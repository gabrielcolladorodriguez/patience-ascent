## Publicar sin MacBook

Tienes Apple Developer (99 USD). **No necesitas Mac:**

👉 Guía paso a paso: **[`PUBLICAR_SIN_MAC.md`](PUBLICAR_SIN_MAC.md)**

Niveles/XP, logros, desafío diario, combos, drag & drop, auto-complete, confeti, hápticos, tienda, monedas.
